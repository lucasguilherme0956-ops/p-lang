# Спецификація языка программированія «П»

*Изданіе первое. Составлено со тщаніемъ и прилежаніемъ.*

---

## I. Общія свѣдѣнія

Языкъ **«П»** есть языкъ программированія высокаго уровня, предназначенный для написанія программъ на русскомъ языкѣ въ благородномъ стилѣ конца XIX столѣтія. Исходный кодъ транспилируется въ Python.

Файлы программъ имѣютъ расширеніе **`.п`**

Кодировка: **UTF-8**

## IV. Синтаксисъ

### 4.1 Объявленіе перемѣнныхъ

```
объявить возрастъ есть 25
объявить имя есть "Ѳёдоръ"
объявить женатъ есть истина
объявить дробь есть 3.14
```

Ключевое слово `объявить` не обязательно при повторномъ присваиваніи:

```
возрастъ есть 26
```

### 4.2 Константы

```
постоянная ПИ есть 3.14159
```

### 4.3 Функціи

```
функція факторіалъ(н):
    буде н не болѣе 1:
        возвратить 1
    возвратить н умножить факторіалъ(н минусъ 1)
```

### 4.4 Условія

```
буде возрастъ болѣе 18:
    печатать("Совершеннолѣтній")
инако буде возрастъ болѣе 14:
    печатать("Отрокъ")
иначе:
    печатать("Дитя")
```

### 4.5 Циклы

```
покуда счётчикъ менѣе 10:
    печатать(счётчикъ)
    счётчикъ есть счётчикъ плюсъ 1

для і въ діапазонъ(10):
    печатать(і)

для каждаго элементъ въ списокъ:
    печатать(элементъ)
```

### 4.6 Классы

```
классъ Человѣкъ:
    создатель(сей, имя, возрастъ):
        сей.имя есть имя
        сей.возрастъ есть возрастъ

    методъ представиться(сей):
        печатать("Меня зовутъ " плюсъ сей.имя)

классъ Студентъ наслѣдуетъ Человѣкъ:
    создатель(сей, имя, возрастъ, университетъ):
        родительскій().создатель(имя, возрастъ)
        сей.университетъ есть университетъ
```

### 4.7 Обработка ошибокъ

```
попытка:
    результатъ есть 10 раздѣлить 0
перехватить ошибка:
    печатать("Произошла ошибка: " плюсъ строка(ошибка))
обязательно:
    печатать("Готово")
```

### 4.8 Модули

```
привезти математика
изъ математика привезти квадратный_корень какъ корень
```

## V. Операторы

Въ языкѣ «П» всѣ операторы записываются русскими словами:

### Присваиваніе

| Операторъ П  | Значеніе              | Аналогъ Python |
|----------------|------------------------|----------------|
| `есть`          | присвоить значеніе    | `=`            |

### Сравненія

| Операторъ П  | Значеніе              | Аналогъ Python |
|----------------|------------------------|----------------|
| `равно`         | равно ли              | `==`           |
| `не равно`      | не равно ли           | `!=`           |
| `менѣе`         | меньше               | `<`            |
| `болѣе`         | больше               | `>`            |
| `не болѣе`      | меньше или равно    | `<=`           |
| `не менѣе`      | больше или равно    | `>=`           |

### Арифметическіе

| Операторъ П    | Значеніе               | Аналогъ Python |
|-----------------|-------------------------|----------------|
| `плюсъ`          | сложеніе               | `+`            |
| `минусъ`         | вычитаніе             | `-`            |
| `умножить`      | умноженіе             | `*`            |
| `раздѣлить`     | дѣленіе               | `/`            |
| `нацѣло`         | цѣлочисленное дѣленіе | `//`           |
| `остатокъ`       | остатокъ отъ дѣленія   | `%`            |
| `въ степени`    | возведеніе въ степень | `**`           |

### Логическіе
`и` `или` `не`

## VI. Правила именованія

- Идентификаторы пишутся кириллицей или латиницей
- Начинаются съ буквы или символа `_`
- Могутъ содержать буквы, цифры и `_`
- Регистръ имѣетъ значеніе: `Возрастъ` и `возрастъ` — различныя имена
- Рекомендуется использовать кириллическіе имена для слѣдованія духу языка
