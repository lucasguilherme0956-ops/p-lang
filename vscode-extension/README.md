# Языкъ «П» — расширеніе для VS Code

Поддержка языка программированія **«П»** въ Visual Studio Code.

## Возможности

- **Подсвѣтка синтаксиса** — ключевыя слова, строки, числа, комментаріи, функціи
- **Иконка файловъ `.п`** — бордовая иконка съ буквой «П» въ проводникѣ (работаетъ съ любой темой иконокъ)
- **Сворачиваніе блоковъ** — `буде...конецъ`, `функція...конецъ`
- **Автозакрытіе скобокъ** — `()`, `[]`, `{}`, `«»`

## Установка

### Изъ VSIX-файла

```bash
npm install -g @vscode/vsce
cd vscode-extension
vsce package
code --install-extension p-lang-1.1.0.vsix
```

### Для разработки

1. Скопируйте папку `vscode-extension` въ `~/.vscode/extensions/p-lang`
2. Перезапустите VS Code
3. Файлы `.п` автоматически получатъ иконку и подсвѣтку

## Примѣръ подсвѣтки

```
объявить имя есть "Міръ"
печатать("Привѣтъ, " плюсъ имя)

функція факторіалъ(н):
    буде н не болѣе 1:
        возвратить 1
    конецъ
    возвратить н умножить факторіалъ(н минусъ 1)
конецъ
```
