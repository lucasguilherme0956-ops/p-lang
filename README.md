## 🔮 Что такое «П»?

«П» — это полноцѣнный языкъ программированія, въ которомъ **все ключевыя слова, функціи и операторы** написаны по-русски въ дореформенной орѳографіи. Это не шутка и не игрушка — это работающій инструментъ съ:

- **Транспайлеромъ** П → Python (CLI) и П → JavaScript (браузеръ)
- **Браузерной IDE** «Писарня» съ подсвѣткой, автодополненіемъ и REPL
- **Расширеніемъ VS Code** для полноцѣнной разработки
- **Словарёмъ** на 7 500 словъ для облагороженія текста
- **20+ встроенными методами** для строкъ, списковъ, словарей и множествъ

## ✨ Примѣры

### Привѣтствіе
```
печатать("Привѣтъ, міръ!")
```

### Функція
```
функція факторіалъ(н):
    буде н не болѣе 1:
        возвратить 1
    возвратить н умножить факторіалъ(н минусъ 1)

для і въ діапазонъ(1, 11):
    печатать(строка(і) плюсъ "! = " плюсъ строка(факторіалъ(і)))
```

### Классъ и наслѣдованіе
```
классъ Животное:
    создатель(сей, имя):
        сей.имя есть имя
    методъ голосъ(сей):
        возвратить "..."

классъ Котъ(Животное):
    создатель(сей, имя, порода):
        родитель(имя)
        сей.порода есть порода
    методъ голосъ(сей):
        возвратить "Мяу!"

объявить котъ есть Котъ("Барсикъ", "Сибирскій")
печатать(котъ.голосъ())
```

### Строки, списки, словари
```
# Строковыя методы
объявить текстъ есть "Привѣтъ, міръ!"
печатать(текстъ.заглавныя())         # ПРИВѢТЪ, МІРЪ!
печатать(текстъ.содержитъ("міръ"))   # истина
печатать(текстъ.раздѣлить(", "))     # ["Привѣтъ", "міръ!"]

# Списки
объявить а есть [3, 1, 4, 1, 5]
а.добавить(9)
а.сортировать()
печатать(а)                          # [1, 1, 3, 4, 5, 9]

# Словари
объявить д есть {"имя": "Пушкинъ", "годъ": 1799}
печатать(д.ключи())                  # ["имя", "годъ"]
печатать(д.значенія())               # ["Пушкинъ", 1799]
```

### Подключеніе библіотекъ
```
привезти математику
печатать(корень(144))     # 12
печатать(округлить(пи))   # 3

# Внѣшнія CDN-библіотеки (въ Писарнѣ)
привезти "https://cdn.jsdelivr.net/npm/lodash@4/lodash.min.js"
```

### Интерактивный вводъ
```
объявить имя есть вводъ("Какъ Васъ звать? ")
печатать("Добро пожаловать, " плюсъ имя плюсъ "!")
```

## 🤖 Создано нейросѣтью

Весь проектъ «П» — языкъ, транспайлеръ, IDE «Писарня», расширеніе VS Code, словарь на 7 500 словъ, спецификація, тесты и документація — **цѣликомъ созданъ искусственнымъ интеллектомъ [Claude Opus 4.6](https://anthropic.com)** (Anthropic) въ діалогѣ съ человѣкомъ.

Ни одной строки кода не было написано вручную. Нейросѣть спроектировала архитектуру, реализовала транспайлеръ съ regex-движкомъ, построила браузерную IDE на CodeMirror 6, создала грамматику TextMate для VS Code, составила словарь дореформенной орѳографіи и написала всю документацію.

Это доказательство того, что AI уже сегодня способенъ создавать полноцѣнные программные продукты — отъ идеи до публикаціи.
